## 💻 Installation & Running

### Option 1: Static Files (No Backend)
Simply open `public/index.html` in your browser.

### Option 2: With Node.js Backend

```bash
# Navigate to backend folder
cd backend

# Install dependencies
npm install

# Start server
npm start
```

Server will run at: **http://localhost:3000**

## 🌐 Live Deployment

To deploy this website live:

1. **Frontend Only (Netlify/Vercel/GitHub Pages):**
   - Upload all files from `public/` folder
   - Works immediately with localStorage

2. **Full Stack (Railway/Render/Heroku):**
   - Deploy the entire project
   - Backend handles API requests
